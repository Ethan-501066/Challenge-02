# Checklist simple

Pequeña aplicación de checklist (HTML/CSS/JS puro).

Cómo ejecutar:

1. Abrir el archivo [checklist.html](checklist.html) en el navegador (doble clic o "Abrir con...").

Qué hace cada archivo:

- `checklist.html`: estructura HTML y elementos de entrada/listado.
- `checklist.css`: estilos sencillos y limpios.
- `checklist.js`: lógica para agregar, marcar y eliminar tareas. Usa `localStorage` para mantener las tareas entre recargas.
