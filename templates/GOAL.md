GOAL
Desired Outcome

Al finalizar el proyecto deberá existir una página web funcional para una florería en línea que muestre sus productos y permita recibir y gestionar pedidos mediante un formulario digital de entrada única, conectado a una base de datos centralizada, eliminando la necesidad de reescribir información de forma manual.

Why This Matters

Este proyecto es importante porque actualmente la gestión de pedidos depende de procesos manuales realizados por una de las socias, lo que provoca pérdida de tiempo, duplicación de esfuerzos y mayor riesgo de errores. Implementar un sistema digital optimiza la gestión de pedidos, facilita el trabajo interno y permite a la florería contar con un punto de venta virtual, aun sin tener un local físico.

Success Criteria

Sabremos que el proyecto fue exitoso cuando:

La página web muestre claramente los productos disponibles.

Los clientes puedan realizar pedidos directamente desde la web.

La información del pedido y del cliente se registre automáticamente en una base de datos centralizada.

Se elimine la reescritura manual de datos.

La socia encargada pueda gestionar los pedidos de forma más rápida, clara y eficiente.
